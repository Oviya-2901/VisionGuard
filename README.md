# 🚀 Overview

VisionGuard is a modern Computer Vision based Laboratory Safety Inspection System developed to improve laboratory safety through intelligent image analysis.

The system enables users to upload laboratory images and automatically performs a comprehensive safety inspection by detecting laboratory equipment, personal protective equipment (PPE), potential hazards, compliance issues, and overall laboratory safety conditions.

VisionGuard provides a modern dashboard with inspection reports, safety scores, compliance monitoring, analytics, and user management in an intuitive interface.
